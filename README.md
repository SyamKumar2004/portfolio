# Syamkumar Potnuru - Portfolio

Welcome to the personal portfolio website of **Syamkumar Potnuru**, a dynamic Data Engineer specializing in building robust ETL pipelines and cloud infrastructure on GCP.

## About the Project
This repository contains the source code for my personal portfolio. It is designed to showcase my technical skills, professional experience, projects, and services.

## Features
- **Responsive Design**: Ensuring a seamless experience across all devices.
- **Modern UI**: Dark-themed, animated interface for a premium look and feel.
- **Sections**: 
  - **Home**: Introduction and quick links to social profiles.
  - **About**: Detailed professional background.
  - **Skills**: Technical and professional skills with progress indicators.
  - **Projects**: Highlights of key data engineering projects (TD & BQ Schema Validation, XML/Parquet File Validation).
  - **Services**: Overview of offerings like Cloud Data Architecture, Automations in Python, and SQL analysis.
  - **Contact**: Get in touch directly via email, phone, or social platforms.

## Folder Structure
